## Ubuntu/Termux
if using termux, maybe multiple plugins it's disabled

### Install
```bash
apt update -y && apt upgrade -y
apt install $(cat requirement.txt)
npm install -g ts-node nodemon

git clone https://github.com/DentaCH/dxtstd-bot
cd dxtstd-bot
npm install
```

### Run
```bash
npm start
```
